# 身近なテーマで実用的なAIチャットボットを作ろう

このパートでは、これまで作った仕組みをベースに、**自分の身近なテーマ**に置き換えて「ナレッジを元に回答するBot」を作ります。  
目標は、**アイデアをカタチにする楽しさ**と、**作って動かす感覚**をつかむことです。  
時間があれば **リッチメニュー** まで整えて、見た目を仕上げましょう。

## 1. このパートで作るもの

自分の身近な困りごとをナレッジを元に解決するチャットボット

## 2. 内容をワークシートにまとめよう

- 制作内容を[ワークシート](https://docs.google.com/spreadsheets/d/1oLxuu5APnb8F0h7VE1wiXt8Ad9AW5i1qbG5wjdJ7qPc/edit?gid=1943902863#gid=1943902863)にまとめながら、これまで学んだことを活かして自由に作ってみましょう！
  - `読み込ませるナレッジ`など、今すぐに決まらない項目は作り終えた後に記入する形でも問題ありません。

## 3. 制作の進め方

- 決まった内容からワークシートに各自埋めていく
- 自分の身の回りの「困りごと」や「こうなったら便利」を1つ選ぶ

## 4. 制作物について

- 選んだテーマでチャットボットを作る
  - 例：Q&A型 / 案内・提案型 / 手順サポート型 / 文章作成・添削型 など
- **RAG（ナレッジ）** を使って、ナレッジを元に答えられるようにする
- チャットボットとして動く状態にする
  - LINE Botにしたい人はLINEと連携する

## 5. 時間が余った人
- リッチメニューで使いやすくしてみましょう。
- 各ハンズオン教材にあるチャレンジ課題をやってみましょう。
- アイデアや制作で困ったら、どんな小さなことでも遠慮なく講師やTAに聞いてみましょう！
